Exercício Condicionais em JS

Exibir uma das mensagens com base na velocidade escolhida:

Se for menor do que 0km/s: "Nave está parada. Considere partir e aumentar a velocidade."
Se for menor do que 40km/s: "Você está devagar, podemos aumentar mais."
Se for maior ou igual a 40km/s: "Parece uma boa velocidade para manter."
Se for maior ou igual a 80km/s: "Velocidade alta. Considere diminuir."
Se for maior ou igual a 100km/s: "Velocidade perigosa. Controle automático forçado."
Imprimir no fim o nome do piloto com a velocidade atual.
